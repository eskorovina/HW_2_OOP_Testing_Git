# Название и краткое описание проекта:
Домашняя работа №2 по теме ООП
Проект позволяет на основе различных рецептов, которые хотят приготовить, формировать список покупок с автоматическим суммированием ингредиентов (т.е. они не будут повтряться в списке), рецепты можно делать на несколько порций, от этого будет изменяться необходимое количесвто покупаемых ингредиентов

# Использование
Как запускать код:
Надо будет при создании исполняемого кода импортировать класса, которые здесь реализованы и воспользоваться методами, которые указаны в классах:
- можно создавать ингредиент
- создавать рецепт
- создавать список покупок + добавлять рецепт

Как запускать тесты:
Необходимо назодясь в HW_2_OOP_TESTING_GIT и НЕ в какой папке, открыть терминал (Terminal - new Terminal) и в открывшемся окне написать pytest
